# Icons Directory

This directory contains icon files for the application.

## Required Icons

- `eye_icon.ico` - Main application icon (Windows)
- `eye_icon.png` - Application icon (macOS/Linux)
- `tray_icon.png` - System tray icon (all platforms)

## Icon Specifications

### Windows (.ico)

- **Sizes**: 16x16, 32x32, 48x48, 256x256
- **Format**: ICO (multi-resolution)
- **Color Depth**: 32-bit with alpha channel

### macOS/Linux (.png)

- **Size**: 512x512 or 1024x1024
- **Format**: PNG with transparency
- **Color Depth**: 32-bit RGBA

### System Tray

- **Size**: 16x16, 32x32
- **Format**: PNG with transparency
- **Style**: Monochrome or simple colors

## Current Status

The application includes a default eye icon generated programmatically.

To use custom icons:

1. Create icons following specifications above
2. Place them in this directory with correct names
3. Restart the application

## Design Guidelines

- Use eye-related imagery
- Keep it simple and recognizable at small sizes
- Use blue/green colors (eye care theme)
- Ensure good contrast for visibility
- Test at all sizes (16x16 to 512x512)
